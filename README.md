# Tsehay Mehari
Premium Habesha Dress Rental & Shopping

### Overview

Tsehay Mehari is a full-stack web application developed to modernize the experience of renting and purchasing Ethiopian traditional clothing. The platform allows customers to browse premium Habesha dresses, search products, add items to a wishlist or shopping cart, and rent or purchase dresses through a simple and user-friendly interface.

The application also includes secure user authentication, product management, and PostgreSQL database integration, making it suitable for both customers and administrators.


## How the Website Works

1. Users visit the home page and explore available Habesha dresses.

2. Customers can browse the collection using the search feature.

3. Users can register and log into their accounts.

4. Products can be added to the shopping cart or wishlist.

5. Customers can rent a dress by selecting a rental date.

6. The system automatically calculates the return date based on a five-day rental period.

7. Administrators can manage products using CRUD operations.

8. All application data is managed through PostgreSQL and tested using Postman APIs.
