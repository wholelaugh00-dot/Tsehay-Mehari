# Tsehay Mehari
Premium Habesha Dress Rental & Shopping

## Description

Tsehay Mehari is a luxury Ethiopian traditional clothing platform that allows customers to purchase and rent premium Habesha dresses online.

## Technologies

- HTML5
- CSS3
- JavaScript
- Node.js
- Express.js
- PostgreSQL

## Features

- User Authentication
- JWT
- Password Hashing
- Shopping Cart
- Wishlist
- Rental System
- Purchase System
- Admin Dashboard
- Shop Dashboard
- CRUD Operations
